Para poder iniciar el proyecto tendrás que ingresar por consola primeramente "npm i" o "npm install" para instalar las
dependencias, luego ingresar "npm start" para correr el proyecto en el navegador.
Este es un e-commerce de venta de diseños, estampados sobre diversos productos. 
El sitio web, está creado con React, es una SPA (single page application), podrás navegar por las diferentes rutas
y componentes de forma más optimizada. Actualmente se encuentra en desarrollo.
